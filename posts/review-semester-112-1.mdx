<br/>

根據這些目標管理時間和個人健康。或許是有更明確的目標，儘管這學期碰到了很多困難的事，有一陣子焦慮失眠還瘋狂掉髮，不過，因為是出於個人意願，總是能喜歡做著這些事(當然還有些鳥事無解囉🙂)。

## **學期回顧**

碩一的第一學期結束了，這短短三個月起起伏伏的過程，我也對「自己想要得到什麼」有更深入的理解。

這件事我沒有侷限在研究所這段期間，因為當初念研究所的初衷之一，便是要學會生活、成就真實自己。那麼關注的核心便不是幾年畢業、選涼還是不涼等這類問題了。

在這學期結束的當下我理解到我真正需要與追求的是什麼：
<br/>
#1. 我的思考與組織需要不斷地被逼到極限。<br/>
不斷地問為什麼、構築完整具說服力的體系（一篇論文、一門技術）。我最快樂的時後是針對問題進行回答、根據他人的想法進行思考再組織思緒的過程，儘管並非所有人皆熱衷於討論，另外一個最快樂的時光就是思考論文相關的事情。不斷地問：這個問題如此回答是否有理？這麼回答有沒有漏洞？這麼回答是否脈絡過於跳躍難以銜接？如果是給回饋的老師的話，會針對什麼提出質疑？
<br/>
#2. 我需要回饋的理由是滿足感，一方面是來自針對疑惑在腦中得以持續整合思考的過程，另一方面是在得知能夠獲得回饋時、具有鼓舞力量的預期心理（我知道我可以做到，為此我必需盡我所能的逼自己到極限）。
<br/>
#3. 我需要深度思考感興趣主題的的個人空間與時間自由。<br/>
<br/>
然後也注意到我自己不良的反應和傾向：
<br/>
#1. 我浪費太多時間在反芻他人的情緒，然而我只是太害怕得罪別人，卻把自己當作無所謂、不重要的（不存在的第三方）。
<br/>
#2. 我浪費太多時間去考慮和組織細微或過長遠的事情，然而這不過是浪費時間，浪費時間規劃切割，結果卻完全沒有照著規劃走。
<br/>
#3. 我很難斷捨離，雖然已經比以前好多了，但仍舊在取捨時游移不決、拖到最後一刻。<br/>
<br/>
## **如果以這學期為單位，我的高光時刻(highlight)會是什麼呢？**

我沒有辦法在第一時間想到，因為總覺得不錯的高光時刻卻缺了點什麼。如果高光時刻的單位不是事件或結果的話，而是行為呢？那我可以說是拒絕他人吧。像是拒絕邀約、準時上下班、有限度的回家頻率，就結果來看，這些是我確確實實地實現了。這幾項拒絕理由都是基於個人需要充足的個人時間（在多數他人看來或許有點太多），儘管很不習慣，還可能回家糾結好一陣子，但選擇的當下還是有記著這點、事後也覺得有這麼做真是太好了。
<br/>
## **想要再更進一步的實踐**

#1. 計畫趕不上變化，將最低限度的規劃做到極致與彈性，比起根據事件劃分，應該使用工作流程的觀點去安排與進行事情；還有，拒絕機會但利用機會、把握機會但活用機會，讓一件事可以達成多重目的。
<br/>
#2. 今年時時提醒自己的一句話：自己和自己比較。這是對降低焦慮、冷靜客觀分析和行動的好句子，對於找回生活控制感是個很有效的咒語；忙碌也會讓人目光狹隘或無心顧及身旁的情況，這時候就會想到大學時期採訪某協會理事長時聽到的一個概念，這個概念沒有被寫進採訪文章中，但對我來說是最重要的收穫：三十歲之前要花時間跟自己獨處。了解自己對個人未來是重要的基礎，而這需要花好多年時間，所以給自己多點時間和耐心。
<br/>
#3. 心態是行動的基準，而行動的操作，個人在2023年學到的重要概念是最小可行性模型與迭代。最小可行性是商業行銷領域中的重要概念，也可以應用於個人生產力、習慣養成等，簡單來說就是將任務拆解成數個小任務，會更容易持續執行、階段性獎勵也能帶來成就感，同時也方便驗證與修正，而迭代類似的流程如PDCA(Plan,Do,Check,Act)，透過重複修正與行動，逐漸推進或優化一件事。
<br/>
#4. 合作、溝通、問問題。自己比較擅長獨自思考和做事情，一碰到和人互動總是猶豫不決、很害怕也因此想太多。碩一上是社交復健期，給了自己小小的目標：主動和他人交流，大概從2023年中開始，從不敢和同學同事打招呼、或主動聊天，到年末，比較敢主動交流了，算是有達標了，接下來就是學習合作和溝通了…這我真的不太曉得該如何閱讀或查資料來準備了，果然還是得要實戰。
<br/>
#5. 選擇果斷、多不一定好，還要少而精。有許多學習的機會，然而時間有限、精力有限，必然需要放棄些什麼。什麼都想要，是一直以來的不良習慣，期望在下學期能夠更重質不那麼重量，在各個事情上能有更多進一步發展思考連結的機會。
<br/>

## **總結**

下學期希望達成寫作輸出、有效溝通、果斷取捨，三個大項目，實踐機會(Plan & Do)包含課程討論、小組專題、個人部落格、工作和學習筆記，然後透過參加活動交流(Check & Act)，如研討會、分享會等，調整和反思行動目標與意義。
