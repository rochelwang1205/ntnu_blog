## **前言**

在開學前，為自己碩士期間訂了幾項大目標，如下：
<br/>

#1. 學會生活、照顧自己：覺察自己的感受、拆解帶來感受的問題並客觀地思考、一步一步持續地練習、改變
<br/>
#2. 建立自學模式 for 長期學習：學習策略、時間管理、閱讀理解與整合能力
<br/>
#3. open mind：be kind and friendly
<br/>
#4. 硬實力：前端工程師

<br/>

而在上學期結束，也發現：<br/>

1. 我的思考與組織需要不斷地被逼到極限。  
    不斷地問為什麼、構築完整具說服力的體系（一篇論文、一門技術）。我最快樂的時後是針對問題進行回答、根據他人的想法進行思考再組織思緒的過程，儘管並非所有人皆熱衷於討論，另外一個最快樂的時光就是思考論文相關的事情。不斷地問：這個問題如此回答是否有理？這麼回答有沒有漏洞？這麼回答是否脈絡過於跳躍難以銜接？如果是給回饋的老師的話，會針對什麼提出質疑？  
<br/>    
2. 我需要回饋的理由是滿足感，一方面是來自針對疑惑在腦中得以持續整合思考的過程，另一方面是在得知能夠獲得回饋時、具有鼓舞力量的預期心理（我知道我可以做到，為此我必需盡我所能的逼自己到極限）。
<br/>
3. 我需要深度思考感興趣主題的的個人空間與時間自由。
<br/>
然後也注意到我自己不良的反應和傾向：
<br/>
1. 我浪費太多時間在反芻他人的情緒，然而我只是太害怕得罪別人，卻把自己當作無所謂、不重要的（不存在的第三方）。<br/>
2. 我浪費太多時間去考慮和組織細微或過長遠的事情，然而這不過是浪費時間，浪費時間規劃切割，結果卻完全沒有照著規劃走。<br/>
3. 我很難斷捨離，雖然已經比以前好多了，但仍舊在取捨時游移不決、拖到最後一刻。
<br/>

## **學期課業標準與目標**

這周開學了解各科目的課程要求及課堂風格，包含有多少報告、小組或個人、文本內容類型等。量化來說，總共有2個團體報告，3篇個人文本導讀，5篇論文提案和完整小論文(其中一篇需全英文撰寫、兩篇為團體)。<br/>

經過簡單的爬梳，對我來說最具有挑戰性的是一門全英文授課的專題課。<br/>

這門課的課程文本對我來說不會太困難、也不是我沒有興趣的主題，但需要英文進行討論對話對我來說壓力很大也很無措，尤其是這周第一堂上課，在開口自我介紹前我覺得我還行，但卻還是緊張到忘詞、講很快。後來想想，有兩個主要原因，一是社交焦慮相關的，二是沒底子和充足經驗。這個問題顯然也無法一周內就解決，所以口說的部份我打算藉由接下來日常口說練習，影子跟讀和回音法，慢慢累積改善。短期解方就先透過日常大量聽看英文學術討論相關影片(哈佛、耶魯等等學校課程的影片)，提升語感和反應了。<br/>

藉由個人報告的機會，順便建立和實踐研究流程及方法，也是這學期的關鍵目標。在寒假的時候找到了《研究方法》，其中有詳細的說明，故這學期想根據這本書，以及各位老師提到過的幾項重要研究方法要點，以更有方法邏輯、可持續性的方法做做看。<br/>

上學期期末回顧中，我發現了自己需要被逼迫到極限，來成長、改變，這學期由於沒有了兼職，時間上也變得更充裕，有更多精力可以放在課業上。然而，在這開學第一週，大概評估下來，由外力推動自己改變的機會並不多，這跟我想要改進或達成的目標：英文口說、研究方法，沒有什麼太直接的關係，而更仰賴於額外的自主練習，如利用通勤時間背單字、花十分鐘英文口語跟讀、空閒時間聽學校學術課程、個人研究資料蒐集和閱讀。<br/>

那麼，這學期相比於上學期從課堂中與同學交流互相學習的目標是不一樣的。一方面每個人的研究方向也來越明顯清楚，一方面也是更熟悉了研究所生活，每個人都有不同的習慣喜好。不過，上課時間也占了一週頗多時間，約兩天，所以個人對課堂的目標是什麼呢?
<br/>
## **課堂目標**

社會科學相關研究所的課程到底意義是什麼?對我來說，經過一個寒假，深刻的體會到這是一個很好的推力讓自己動起來，不要再偷懶摸東摸西了。那進一步來說呢? 有不少課程都有團體報告，上學期可以說是促進同學互相認識的機會，那這一學期呢? 課綱將授課核心目標，批判性思考，以團體報告的規定為產出目標。我能理解減輕同學負擔、或者快速但有效的帶學生走過產出篇研究的整個歷程，但我寧可個人寫一篇小論文繳交。說實話，我有點茫然，總覺得能再多做點什麼。
