> 文本：《橫觀東亞：從核心現場重思東亞歷史》<br/>
>第一輯的第一章〈從「核心現場」探索東亞共生之路〉、<br/>
>第二章〈連動的東亞、問題的朝鮮〉、<br/>
>第四章〈複合國家與「近代的雙重課題」—重審東亞二十世紀歷史〉
<br/>

&emsp;白永瑞在《橫觀東亞：從核心現場重思東亞歷史》第一輯的第一章〈從「核心現場」探索東亞共生之路〉、第二章〈連動的東亞、問題的朝鮮〉、第四章〈複合國家與「近代的雙重課題」—重審東亞二十世紀歷史〉，從自身的經驗與做為學者的身分，梳理東亞歷史脈絡與發展，歸結出雙重近代課題：運用雙重周邊視角從核心現場看見東亞未來共生之路的可能性，在這過程東亞各國必須適應近代資本主義建構的體制，同時克服滿布傷痕、複雜因果的近代。為東亞構築宏偉的未來藍圖的同時，也運用南北韓分斷體制的案例—開城工業區，以複合國家為過渡期，試圖從經濟合作的視角，提供可實踐東亞共生的方法。
<br/>

&emsp;人人都是共生的實踐者，在第四章末提出的全球地域學，在課堂中教授提出了易懂的例子：使用環保杯。其概念展現在選擇使用環保杯是因為想到了塑膠帶給全人類、環境生態的長期負面影響，也可說是有推己及人與大同的意味。從這個例子再延伸想，聯合國SDGs永續發展目標的精神也是如此。近幾年來，學校企業積極推動SDGs，包含講座活動、政策規劃等，並將這些作為指標的標準。在此風氣下，我們有許多機會碰到相關的事情。

<br/>

&emsp;大學時期受同學採訪關於使用環保餐具的頻率與使用動機，會被邀訪的原因是：因為參與過羅浮社，辦了海洋教育（塑膠微粒、食用道德）相關的幾場講座、喜歡露營。認為親近自然者就等於會使用環保餐具的假設，主觀來說感覺還挺合理的，但根據同學的採訪結果，像我這樣的受訪者—喜愛自然但不使用環保餐具，並非少數。

<br/>

&emsp;這樣的落差到底源於何處? 儘管需要多付幾塊錢，但通常也不會讓我們少用塑膠製品，因為方便。需要自備及清洗環保餐具的麻煩，是主要因素。

<br/>

&emsp;當現實利益（如目前在多數店家不提供一次性餐具或提袋）與個人習慣（不方便、懶惰等）產生衝突，如「要多花一元購買吸管」還是「自備環保吸管」，在不考慮放棄不買的第三種情況下，最終必然選擇兩者之一，通常習慣會佔上風。現今服務業消費者至上的觀念，個人認為，也是難以落實環保行動的要因素之一。

<br/>

&emsp;如何有效地推動實踐? 作者定義二十一世紀形國家利益的四個標準：（1） 整體性，是國內一部分人的利益，還是全體國民的利益；（2） 持續性，是暫時的利益，還是可持續的利益； （3） 直接性，利益是否可以直接發揮作用?；（4） 共存性，是否能和國際社會的利益並存。作為當代國家利益的標準，既適應近代也克服近代。仿照作者重新定義了國家利益，也可重新定義個人利益。

<br/>

&emsp;在儒家經典中，個人從出生到死亡，所經歷的環境、接觸了人事物都發於個人與歷史長河交融的意識，群體先於個體。具有強烈的環保意識，其中基於推己及人者，在我的經驗中，可說是少之又少。直接宣傳環保愛地球、敦親睦鄰、兄友弟恭、尊老敬賢，顯然不是最適合的方法，有時候還會產生反效果，造成更多衝突與更深的誤解。

<br/>

&emsp;在定義標準前，要先問為甚麼要使用環保餐具? 政策宣導或者相關非營利團體倡導和腦中想到的原因或許相差不遠：因為對人體及環境無害、減少垃圾產生、方便攜帶、有質感（群體認同）。提倡環保固然是好事，但也造成一些未沒有預想到的負面影響：通常，環保餐具部分包含甘蔗渣、回收紙等天然、較易分解的材質，但還是有較耐用、不易分解的成分。所以使用環保餐具雖有許多好處，但也造成如不重複使用的環保餐具，也等於是製造更多垃圾。

<br/>

&emsp;個人利益的標準重構成需在原有的利益和目標間找到平衡。我想標準必須是切身相關，如對個人健康、金錢帶來的損失有感，並且從改變行為獲得效益，如成就感、實際好處等，加上友善環境的次要好處與做為世界公民的身分認同，同時也必須更進一步思考這些目標和行動帶來的後續影響，如是否能落實在生活中、持續使用與重複使用環保餐具等。

<br/>

&emsp;作者提出了美好理想，也從經濟合作的大方向，提出可行與有巨大影響力的例子。要推動東亞共同體與共生，有賴於東亞各方學者的團結。由具有影響國家層級之政策的團體，如作者提的學者群，以長遠發展、公平和平為旨，創造不為短期利益與少數人得利的平等和諧社會。

<br/>

&emsp;閱讀此三篇文本後，我再思考了環保之於自身的意義，並試圖提出了可能的實踐辦法。儘管現實中道德理想常常容易被置於最後，但大同世界的樣貌仍舊讓人心生嚮往，這些或許在日積月累下，終將化為實際行動的巨大動力。
