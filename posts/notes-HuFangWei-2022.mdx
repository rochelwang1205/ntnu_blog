> 文本：大洋洲的區域化進程與重要議題探討，胡方維，分裂的世界? 21世紀全球區域化崛起。<br/>

## OPENING

&emsp;2014年太平洋島國論壇提出了大洋洲區域主義架構，以此為21世紀後檢視大洋洲區域觀念與相關議題發展的重要分水嶺。接下來我們會以2014年為分界討論在2014年前相關的學術研究以及主要議題是什麼。近百年大洋洲重要研究議題圍繞三大項：區域安全、核武議題、海洋議題，內容隨著全球局勢的轉變而有所改變。
<br/>

## 2014前有關大洋洲地區的重要研究議題

### 反殖民、核武議題

&emsp;二次世界大戰之後到1960年代大洋洲區域，主要還是由原本的宗主國來掌握各個區域的事物，所以大洋洲的表現跟作為主要都是關注南太平洋委員會這個區域組織的建立跟運作。
<br/>
&emsp;而此時期的學者，主要追蹤探討整個組織的運作及其成果，或有學者也會以較大的較廣的層面，例如從戰後統籌重建二戰受害嚴重且無力復興的同盟國參戰國家的聯合國善後救濟署(United Nations Relief and Rehabilitation Administration，UNRRA)去探討到整個大洋州地區的區域安全或是戰略相關的議題，或檢討了後殖民時期區域發展的狀況與相關的相應措施的研究。
<br/>
&emsp;到了1960至1970年代已經有關於民族的區域民族自決反殖民化與地區認同意識的聲音以及南太平洋論壇的成立，這時期已有關於大洋洲地區地理邊界以及地區身分的論述的建構。而到了1970到1980年代，相關研究結合了反殖民風潮及南太平洋論壇的成立進行討論，區域安全成為了學界關注的焦點。關於大部分的相關研究，學界都是從全球戰略的角度來對大洋洲區域進行分析和辯論，仍然是以區域內或是外部的大國在當中區域的戰略利益為考量點出發。如果是從大洋洲內的小島國家或是地區的角度來出發者則多是探討反核的議題。
<br/>
&emsp;反核一直到現在還都是一直被持續討論的一個問題，主要是美國在大洋洲地區的核武試爆引起了諸多的損害跟不滿。這件事情除了在冷戰時期被檢視之外那到了現在被持續探討跟處理，至今仍是未解的一個議題。而中國大陸則利用呼籲大洋洲島國共同維護核武不擴散條約，並且以太平洋無核區條約來批判美國、英國與澳洲展開核子潛艦建造，試圖拉攏太平洋島國的支持，希望能夠建立相應的合作。核武相關事務，從過往早期的軍事戰略安全議題，到了現在經濟、國際法、人權等等層面一直都是大國角力時關鍵議題之一。
<br/>
&emsp;1990年代到21世紀初期冷戰結束，面對不同的國際關係情勢，大洋洲的區域安全問題重新有了新的定義。(1)21世紀初期亞洲國家，特別是中國的影響力開始上升了，構成區域安全問題的內涵發生的改變；(2)而九零年代初期，全球的跨國犯罪問題也逐漸升溫；而(3)2000年之後大洋洲部分國家及地區面臨內部政治的危機；(4)以及後續美國911恐怖事件之後全球反恐戰爭的風潮等等的影響，都促使大洋洲區域的大國澳洲與紐西蘭更積極提議要深化區域整合以處理這些新興安全議題。
<br/>

### 海洋議題
&emsp;大洋洲作為一個一海洋為主的區域，區域內各國家或地區如何在涉及海洋的議題上爭取國家權益，如何在擁有有限技術的情況下能充分利用與管理海洋資源獲取的相關援助一直是區域中的重要關注項目，1973年開始的聯合國第三次海洋法會議歷經九年多漫長的制約談判，終於在1982年通過被稱為海洋憲章的聯合國海洋法公約。大洋洲地區國家或地區如何運用聯合國海洋法公約、如何藉由區域途徑推動海洋事務管理海洋污染設立區域，漁業管理組織或安排及永續利用海洋資源逐漸成為該區域研究的一項重要議題。
<br/>
&emsp;隨著大洋洲區域那個國家或地區或主動或被動的面對去全球化帶來的挑戰大洋洲區域的小島國必須正視其先天的經濟與環境脆弱性。
對全球化的反思促使學界相關研究開始重視探討大洋洲區域哪一個國家或地區作為P SIDS本身的特徵與挑戰，譬如在海洋事務議題方面因小島國自身在經濟與技術方面的弱勢，需要藉由集體外交尋求相應的國際財務援助與技術轉移。PIDF的設立也帶動了大洋洲區域在塑造新的區域秩序時可維護PSIDS在國際場域中自身的利益，這些因素共同推動了大洋洲區域主義在經濟氣候變遷等環境議題以及地緣政治方面的新發展。
<br/>
### 小結
&emsp;總結來說在21世紀之前學界對大洋洲區的研究大致發展出了下列研究議題：
<br/>
&emsp;在高度政治議題方面，(1)區內PSIDS自覺/決與區域意識的興起帶動的區域內外大國與區域內島國間的權力關係的演變，和(2)從冷戰時期關注核武軍事安全以及(3)大國在大洋洲區域運用核武與其相關醫療問題的解決，以及(4)打擊跨國犯罪與海洋執法安全等議題對區域內外大國與區內島國間權力關係之檢視，奠定了今日檢視其他區域外強權如中國大陸介入時的基礎認識。而對核武遺留問題與新興安全議題之討論，則成為如何觀察今日區內外大國間或與區內島國間戰略對抗或合作的分析工具；
<br/>
&emsp;在低度政治議題方面，(1)由大洋洲的區域特質與脆弱性出發，集合太平洋共同體架構下所提出之三計劃六組所包含的議題，(2)學界關注氣候變遷環境永續及(3)海洋相關議題譬如檢視大洋洲區域如何運用國際海洋法公約等國際文件之規範，(4)或該區域中相關漁業管理組織運作，海洋環境保護與海洋資源開發利用之合作，以及(5)在全球化經濟結構下如何推動大洋洲區域經濟發展強調人權社會與教育等之發展的議題。
<br/>
&emsp;上述研究也提供後續大洋洲區域結合海洋特質的基礎，以環境與永續發展為核心發展理念建構出促進該區域經濟發展的藍色經濟概念。
<br/>
## 2014後有關大洋洲地區的重要研究議題
&emsp;2014年除了太平洋島國論壇提出的大洋洲區域主義架構之外，聯合國也舉辦了第三次小島開發中國家國際會議。在會議中指出有鑒於SIDS之特質與特有的脆弱性性質，在克服各項國際環境求證時需要藉由國際合作的途徑，並且在這個會議中產出了一份名為薩摩亞途徑(小島發展中國家快速行動方式)的國際文件。同年11月，聯合國大會決議贊同該會議所產出的文件。
薩摩亞途徑旨在促進小島嶼國家可持續發展，其落實進展每五年接受一次評估。
這些全球與區域階層的發展與內涵形成了2014年後探討大洋洲區域的主要研究方向，其中以氣候變遷議題為核心，檢視區域在集體外交深化區域整合下發展出的區域集團的作為。
<br/>
&emsp;太平洋島國論壇倡導要由區域內小島國家或地區深化區域整合，並建立與區域外國家的對話夥伴關係，同時PIDF所採行的開放性會員資格及PIDF憲章第八條所規定的外部各方，包括發展夥伴技術夥伴等，均顯示大洋洲區域組織尋求區域外夥伴的渴望。因而可能受到更多區域外國家的影響。隨著中國大陸崛起，大洋洲區域在二戰後再次成為東西方，尤其是中國美國太平洋戰略博弈的戰略要地。回應此一國際發展趨勢，大洋洲區域的研究方向也開始出現諸多探討中國大陸崛起與大洋洲區域戰略安全的內容。
<br/>
&emsp;接下來就氣候變遷太平洋小島開發中國家集團與綠色藍色經濟，以及中國大陸崛起與大洋洲區域戰略安全，詳細說明。
<br/>
### 一、氣候變遷太平洋小島開發中國家集團與綠色藍色經濟
&emsp;1992年地球高峰會議正式將永續發展明確化，開始環境自然災害與氣候變遷的議題探討。隨著全球氣候變遷議題之發展「共同但有區別之責任」的原則成為國際環境法中的重要原則。開發中國家，特別是SIDS在應對全球化的經濟發展與環境保護的議題時在外交談判上佔據獨特地位，且掌握其特有談判籌碼的談判集團。
1992年地球高峰會議要求SIDS於聯合國舉行一次小島開發中國家永續發展全球會議，來促進生活在小島開發中國家人民的利益。
<br/>
&emsp;1994年5月於巴貝多召開的首次全球小島開發中國家永續發展會議，通過了《巴貝多宣言》以及《小島開發中國家永續發展聯合國行動方案》，簡稱巴貝多行動方案。在巴貝多宣言中，SIDS認識到自身極易遭受自然環境災害所帶來之損害的特徵，但另一方面SIDS從這些災害中復原的能力卻是有限，同時SIDS又是造成氣候變遷問題起到最少作用的一些國家，且SIDS在推動經濟發展時，尚有諸多制約因素，包含島礁陸地面積狹小、大量依賴專屬經濟海域帶來的海洋資源、以及多依賴發展漁業或觀光業、難以發展工業等狀況。
<br/>
&emsp;
<br/>
&emsp;相關研究也指出儘管各個小島國家間存在差異，但總體而言SIDS 仍被認為是最容易受到氣候變遷影響的一個特殊群體，為此需要全面檢視SIDS的困境與跨學科解決方案，因此SIDS認識到自身需要利用國際間所認可的「共同但有區別之責任」原則以尋求國際社會的援助與支持，同時SIDS之間需要藉由合作及建立互助夥伴關係，以落實所擬定的行動方案，並進一步在國際社會的援助與SIDS的合作之間建構所需的法規體制與資訊分享網絡。
<br/>
&emsp;2014年於薩摩亞首都阿皮亞所舉行之第三次小島開發中國家國際會議，延續之前的發展，以SIDS特徵與特有的脆弱性，提出SIDS應重視國際合作途徑，在薩摩亞途徑此一國際文件的指導下建立各項事物的合作機制。

在此概念下太平洋地區的SIDS進一步建構出太平洋區域集團太平洋小島開發中國家的概念，14個PSIDS集團的會員國不僅是太平洋共同體的成員，PSIDS本身並已成為聯合國階層多邊談判場域中之談判集團。大洋洲區域小島國家或地區以PSIDS集團作為回應聯合國永續發展目標的重要談判集團對象，持續在不同的國際會議上也以SIDS集團的身分提出其主張。

2012年，PSIDS集團各國領導人在斐濟召開與太平洋接軌會議，贊同召開太平洋島嶼發展論壇以讓關鍵部門的領導人參與PSIDS綠色經濟政策的實施。
<br/>
&emsp;在PSIDS落實綠色經濟就等於是要落實藍色經濟。在2012年召開聯合國永續發展會議之前2011年第二次籌備會議上PSIDS集團集體發表聲明指出對PSID集團而言綠色經濟事實上就是藍色經濟，所以在PSIDS集團的觀點中海洋漁業議題必須在里約+20的議程中被突顯出來。

PIDF官方網站介紹自身目標之一，是為綠色藍色的太平洋經濟提供有利環境，其中闡述的綠色經濟之中，只是為所有人帶來永續發展公平與福祉，並具被包容參與與課責性，以提供永續的消費與生產，而藍色經濟就是遵循所有綠色經濟之原則必將之落實於海洋場域中。PIDF介紹太平洋綠色藍色經濟模型時，開宗明義提到PIDF成員之管轄範圍中有超過99%都是海洋，所以在討論綠色經濟時，必然會提到藍色經濟以彰顯太平洋的實際情況。由此建立的模型中不論是綠色經濟或是藍色經濟皆是由經濟發展、環境保護與社會公平三個支柱支撐起的發展原則。隨後PIDF也於2017年在索羅門群島由所羅門政府主持召開了「第一次高階太平洋藍色經濟會議」。

而在此一連串的行動中，PSIDS集團逐漸形成了一項共識：擁有廣大海域的國家需要將自身視為藍色經濟體最大程度的提高藍色經濟帶來的回報，同時遵循永續發展的原則。
<br/>
&emsp;這樣的發展也促使在2014年後研究大洋洲區域時在低度政治議題方面則必須多關注區內國家或地區及PSIDS集團如何藉由區域途徑處理氣候變遷綠色藍色經濟與永續發展的議題。
<br/>






### 二、中國大陸崛起與大洋洲區域戰略安全
&emsp;大洋洲區域在軍事經濟等事務上的安排一直以來受到區域內外大國戰略佈局考量的影響。2011年時任美國國務卿的希拉蕊，曾表示中美兩國處於直接競爭關係的主要原因是來自於中國大陸在南太平洋積極拓展影響力的作為。中國大陸除藉由經濟貿易措施積極爭取與大洋洲區域內國家建交外，在軍事層面上也強勢意圖於大洋洲區域內駐軍與建立軍事基地。大陸國務委員兼外長王毅出訪南太平洋多國的行為也進一步引發美澳等國的反彈。中國大陸在大洋洲區域的積極作為引起區域內外大國，尤其是美國與澳洲的警惕。
<br/>
&emsp;從援助金額也可見中國於大洋洲地區的野心，從澳洲智庫Lowy Institute提供的圖表數據可以看到，以大洋洲援助金額排名來看，截至2021年，澳洲排名第一，中國第二，之後依序為美國、日本、紐西蘭。中國自從 2006 以後，已成為大洋洲地區中主要的援助國，通過援助計畫以換取政治影響力（Pryke 2018）(郭佩宜、李政政、Sra Manpo Ciwidian，2022)，其中各國援助還是借貸的形式也值得關注。中國以借款(lend)的方式、美國等國是以捐款(grant)，兩種方式各有優缺點，借款的好處是快速方便，但需要同意部分條件，以及可能面臨未來的債務危機；捐款形式則是不用償還，但卻需要相對繁瑣的審核程序才能拿到。前者有令人擔憂的債務危機，後者的條件規範則也存有是否是新殖民主義。
<br/>
&emsp;近期一件事或許多少能體現兩種援助方式的差異。今年2月中旬傳出帛琉現任總統惠恕仁(Surangel Whipps Jr.)致函美議員，透露中國將以「訂滿帛琉所有的飯店房間」為由誘使帛琉與台灣斷交，信中也寫到美方急需通過自由加盟協約（Compact of Free Association，COFA）援助款一事。
COFA是美國與密克羅尼西亞聯邦、馬紹爾群島(目前為台灣邦交國)以及帛琉(目前為台灣邦交國)三個太平洋島國簽署的國際條約，確立並制定了彼此間聯繫國的關係，美國向三國提供經濟與安全的援助。
隨後，我國副總統賴清德與惠恕仁進行視訊會議強調將持續對帛琉提供具體援助。
中國政府於2017年11月下令禁止中國旅行社出團到帛琉，但在2023年又重新開放，並重啟澳門包機航班，中資先前在帛琉閒置的房地產亦開始重新整修。中國對帛琉是否會在11月選前加強經濟利益的磁吸力道，進而影響我國與帛琉的邦誼，值得持續關注。
<br/>
&emsp;針對COFA援助款，美國於三月初簽署確定援助款項，大抵來說程序有了進展。內容指出20年的期限和71億美元的強制性資金承諾。
大洋洲內外大國對中國大陸的態度以及大國與大洋洲區域內島國的互動逐漸成為關注焦點例如美國、英國、日本、澳洲與紐西蘭等五國藉由組成藍色太平洋夥伴聯盟，以及有美國、日本、印度與澳洲四國參與的四方安全對話，利用共同應對氣候變遷或生存安全等議題與大洋洲個島國建立更堅實的經濟與外交合作，以此對抗中國大陸在大洋洲區域的野心。
<br/>
&emsp;夥伴國家指在2023年9月22日藍色太平洋夥伴聯盟外長會議聯合聲明(Joint Statement on the Partners in the Blue Pacific Foreign Ministers Meeting)的國家，另外包含了加拿大、德國及南韓。
原本在21世紀初期逐漸淡出大洋洲區域研究的核武問題，也再度浮上台面成為美中拉攏區內島國的籌碼。
大洋洲區域與二戰中展現出對美國總體戰略佈局的重要性，中國大陸想要成為世界強權，急需突破美國在大洋洲的佈局及對亞太帶來的影響與箝制。除了美國印太戰略所展現的圍堵外，美國議會利用其他議題或政策工具，如打擊IUU及相隨勞工虐待國安備忘錄所顯示之打擊非法漁捕等漁業資源養護或海洋環境保護等方面，制約中國大陸日漸強大的影響力，隨著情勢發展相關研究也更重視中國大陸在大洋洲區域發揮更強大影響力的可能。
除了美國擔憂自身以大洋洲區域為跳板進入或說重返亞洲的路徑受阻，處於中國大陸與大洋洲之間作為大洋洲地區主導大國的澳洲也擔心自身試圖維繫的大洋洲區域秩序受到挑戰，譬如索羅門群島計劃與中國大陸展開安全合作就引起了歐洲的緊張。
<br/>
&emsp;中國大陸崛起來的情勢變化，促使大洋洲區域研究，再次開始探究大國在區域方面的戰略與作為以大洋洲區域內外大國為核心的相關研究，研究主要集中在美、中、澳三方的雙邊與多邊合作或競爭關係，基本以美國與澳洲如何遏止中國大陸勢力強勢介入大洋洲區域為主。
<br/>
&emsp;儘管澳洲與紐西蘭還在大洋洲區域內保持主導地位，但中國大陸與美國的政策如中國大陸的一帶一路政策以及美國拜登總統上台後延續亞太再平衡策略如於2022年二月份白宮發佈的美國印太戰略文件都可能重新改變大洋洲區域秩序，美澳兩國對抗中國大陸進入大洋洲的作為為該區域建立了一種新的區域秩序樣貌，澳洲強調自身利益的同時會與中國大陸建立合作或對抗關係也成為後續持續關注的議題。
<br/>
&emsp;除了區域內外大國的作為受到區域研究的重視外，大洋洲區域內其他小島國家或地區在該區域大國戰略競爭環境下如何發揮其影響力與能動性也成為大洋洲區域研究的一種方向。如中國大陸試圖與大洋洲10個島國就傳統與非傳統安全問題進行集體合作，如培訓當地警察加強數據通訊與海洋漁業的合作，並建造孔子學院大外宣中國的意識形態等，但在大洋洲部分島國的反對下並未成功，凸顯了區內小國亦能集體抗阻區域外大國勢力的入侵。
<br/>
&emsp;除觀察歐洲與美國等大國之作為外，中國大陸不斷試探大洋洲區域各國對其忍受程度之作為，譬如中國大陸武官企圖混入太平洋島國論壇遭斐濟警方驅離等案例等亦值得觀察。
<br/>
&emsp;(1)大洋洲個小島國能否藉由區域途徑及既有共同決策機制以掌握自身在此區域中的政治話語權並與域外強權抗衡？
(2)或區域內島國之集體決策機制是否會在中國大陸的介入及各個突破下分裂如吉里巴斯曾或因中國大陸提供之誘因而退出太平洋島國論壇？
以上這些均可成為後續研究大洋洲區域安全與戰略平衡等高度政治議題的研究課題。

## 大洋洲對臺灣的重要性
&emsp;今年一月，諾魯發布正式聲明指出，為了諾魯共和國與諾魯人民的最大利益，將轉向一個中國原則（One-China Principle），承認中華人民共和國（People's Republic of China）是代表中國的唯一合法政府，並尋求與中國恢復全面外交關係，代表不再承認中華民國（台灣）為一個獨立國家，即起斷絕與中華民國「外交關係」，不再與中華民國發展任何官方關係與往來。諾魯曾於2002年與中華民國斷交，轉與中國大陸建交，不過又在2005年5月14日與我國復交。
