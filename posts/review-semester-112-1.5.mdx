## **抉擇過程的紀錄**

寒假正好讀到了《大學不迷茫：願你的青春無所畏懼，亦不留遺憾》。比起相見恨晚、不如說來的是時候的一本書。裡頭為大學生從懵懂出入校園到即將畢業步入社會的新鮮人的種種疑惑提出精確又實用的指引，使迷茫心靈行動有所方向。從校園生活、社團、宿舍、課業、工讀到戀愛、人際方方面面，基本涵括了大學學生會有的困惑。
<br/>
雖說早就離開了大學，但還是有幾點最有感：不管他人眼光堅持做自己以及不要為了小錢放棄學習和成長的機會，畢竟時間過了就沒了。
我是個非常在意他人眼光的人、也很受他人影響，所以書中提到堅持自己對我來說很困難，但大學時期我太晚想到搬出去一個人住或者回家住的解方—還是因為怕麻煩家人接送（既沒車、通勤也不方便）。所以睡也睡不好、生活過得戰戰兢兢，四年精神好的日子用手指都能算出來…也正因為如此在校名和個人空間我選擇了離家遠點能夠租房（能跟家人拿錢拿得比較安心）。
<br/>
「自己賺的錢」這個執念從小就根植於心，總覺得有了才安心、也能自由地選擇拿這筆錢做什麼，所以考研究所到考上研究所後學習寫網頁的課程都是拿了兼職的錢去做，我當作是能夠為自己負責選擇想做什麼的要求，我也因此準備考試、認真學習寫網頁，但也難以割捨顯然對目前的我帶來成長的兼職工作。
<br/>
在離職和留下來之間擺盪了好幾個月，閲讀了李尚龍老師這本書後也才是算下定決心要作出改變了。
<br/>
先來說說考量的點。想要離職的念頭已經考慮了兩個多月了，期間反反覆覆。
去還是留，各自主要的論點都讓我很糾結，但主要都是從時間有限的觀點出發。
留下的主要論點在於，身為非本科、基本上也算轉職工程師，又根據一年不到與許多人交流的結果來看，經驗與能力是這個職業最看重的點，對我來說目前最好的方法是從現在的兼職累積年資和經驗(而且公司這麼好!同事人也很好!)
<br/>
離開的原因在於，工作兩個月之後，我對於進辦公室上班有強烈的厭惡排斥感。撇除辦公室環境比較壅擠、沒有個人空間等因素，因為兼職無法處理較大、複雜、緊急的任務。這也涉及到公司目前發展情況比較需要即戰力與全職投入的人。在這過程，無法確實扎實的學習的代價，在事後評估後，我認為遠超過累積年資與實戰經驗帶來的好處。
<br/>
而真正決定離職的最後一根稻草是去北京交流的經驗。這邊就不說明細節，僅略述心情。
<br/>
> 如今，北京的面貌在心中還是如抵達離開首都機場看到的那般風景：相同的高樓層疊如棋盤佇立在黃色土地上，遠看竟生硬冰冷，猶如死木。在廣袤土地上有著萬千相似一致的面孔。國家這巨大的機器由無數的齒輪卡榫精密地、有效地推動著。旅程中、旅程後有個有趣的事情，那就是身邊的人不是叫我重考就是什麼也都不說、但還是對外炫耀我是哪個大學畢業的。對於這兩者的反應我真的哭笑不得。很訝異自己能夠這樣看待。我以為自己會開始內耗、自我糾結。最真切的感受大概就是：畢業後我絕對不要回家。為此要更積極面對自己的感受，還有有計劃地努力。然後我做的第一件事，就是跟公司提離職了。—節錄自個人日記

<br/>
提了離職後，我還是懷疑這是否是個好的選擇。但想到這些時間能夠拿去做我更想做的事情，準備資料、考試、做side project、技術讀書會…心緒也就安定了些。
<br/>
希望幾個月之後我會真心高興做了離職這個決定吧！
<br/>
畢業後，還會不會回到公司、回到該產業是一回事，一切緣分，但我確定還是會在工程師的路上，至於方法與形式也就未知。
