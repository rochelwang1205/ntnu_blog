> 文本：吳玉山，「權力不對稱與兩岸關係研究，（第二章）」，《重新檢視爭辯中的兩岸關係理論》（2009），pp. 31-59。
<br/>

&emsp;吳玉山(2009)以權力不對稱理論中重要的研究論點來檢視兩岸關係，包含Womack的權力不對稱(power asymmetry)、吳玉山的大小政治實體模式、蘇軍瑋從歷史角度的詮釋途徑，以及唐欣偉提出的根據特定政策被選擇後之結果，並說明其他重要研究途徑納入權力不對等之因素後，更能貼近並分析推演兩岸關係的變化。這些重要的分析角度或研究途徑包含分裂國家、外溢效果(spill-over)、賽局理論、國內政治心理、政治經濟學、選票極大化、國際體系、戰略三角。
<br/>

&emsp;Womack、吳玉山、蘇軍瑋三位學者分別以中越關係、蘇聯與鄰近十四個國家關係以及明鄭時期與清廷為例，探討決定大國及小國政策的原因，唐欣偉則假設小國為理性冷靜求生存，探討在特定政策被選定後可能產生的結果。
<br/>

&emsp;在權力不對稱的常態下，Womack認為大小國關係由大國對小國的承認(recognition)加上小國對大國的遵從(deference)所組成，會有這樣的結果，是因為不對稱權力結構影響大小國相互政策的機制，在大國無法使小國屈服以及真正消滅小國下，且小國預期被滅亡的衝突結果外，兩國最終找到與達到遵從和承認的持久性均衡、常態(normalcy)。
<br/>
&emsp;
吳玉山則認為大小國關係是由大國屈服小國的意志此一常數加上小國選擇抗衡(balancing)或扈從(bandwagoning)策略所構成，其中小國會根據相對經濟發展程度和小國在西方所獲得的支持兩項重要變數來決定策略。吳玉山(1997)應用大小政治實體模式於兩岸關係上，得出四個結論：台灣對中國只有抗衡與扈從兩種選擇、兩岸經濟發展程度差距大時，台灣更傾向抗衡策略、若有美國支持，台灣抗衡策略會更明確、台灣內部存在不同策略主張之立場。
<br/>

&emsp;蘇軍瑋採張啟雄之名分秩序論的概念，也就是東方獨有之中原王朝與鄰國互動關係，王朝強盛而鄰國相對虛弱則兩者採懷柔與稱臣，反之雙邊選擇鎮壓與入侵的武力衝突選項。若新王朝鞏固，短期內王朝可能採取懷柔政策以收攏人心。蘇軍瑋以歷史的詮釋，探討在小國曾為大國統治者之情況下，由於歷史與理念兩項因素，小國除了抗衡與扈從的選項外，還有反攻光復(rollback)的選項。
<br/>

&emsp;唐欣偉根據三個雙元變數，提出了八個發展路徑以及小國的五種最終結局。首先是大國侵犯性主權要求的種類，通常大國會選擇羈糜就滿足而非兼併，再者小國有無強盟友，將影響小國傾向讓步或不讓步的選擇。以以色列對亞述、猶大對亞述、魏國對秦國、鄭國對楚國、捷克斯洛伐克對納粹德國、芬蘭對蘇聯六個例子，說明大國多選擇「以夷制夷，因其俗以為治」，而小國多不會選擇與大國開戰，這使小國避免不戰而亡或戰敗面亡的最終結局，而取得勝利自主、妥協或戰敗妥協的結果。根據推斷與特定條件，小國即可反推最理性的策略。
<br/>

&emsp;以上四個理論應用於至馬英九政府時期的兩岸關係的解釋力各有優缺點。情況符合Womack所描述的結果，然而其理論存在權力不對稱結構去解釋雙方政策和關係轉變，也就是常數解釋變數的問題。吳玉山的大小政治實體模式也具有解釋力，即在美國支持下，1990年代中國相對經濟發展超越台灣後，台灣抗衡策略確實也更加明確。從近一百年的兩岸發展來看，蘇軍瑋的解釋符合趨勢，然而卻無法表示未來就沒有衝突。唐欣偉的理論可解釋馬英九政府時期的新大陸政策，也就是在中國採取羈糜的態度且美國仍採一致的支持下，小國最理性的選擇便是與中國妥協。
<br/>

&emsp;若將蔡英文政府時期納入探討理論應用兩岸關係的解釋力，蘇軍瑋無法解釋到的近年兩岸衝突發生與對立升溫，根據唐欣偉的理論，在蔡英文政府時期由於認為中國對台灣採取兼併態度更勝羈糜，若美國為台灣真正盟友，抗衡對立而不求妥協是個合理的選擇。
