<br/>
在進入研究所前，為自己碩士期間訂了幾項大目標，如下：
<br/>

#1. 學會生活、照顧自己：覺察自己的感受、拆解帶來感受的問題並客觀地思考、一步一步持續地練習、改變
<br/>
#2. 建立自學模式 for 長期學習：學習策略、時間管理、閱讀理解與整合能力
<br/>
#3. open mind：be kind and friendly
<br/>
#4. 硬實力：前端工程師

<br/>

而在上學期結束，也發現：
1. 臨場或有提前準備的舉手發言/發問
	>緊張、擔心問題不好、被嘲笑或對主題沒興趣
	>後來就用私下一對一的方式問
<br/>
2. 被針對的感覺
	>總之，後來也用諸多的方式來化解這個不舒服的感覺，像是念課外書、思考(內耗??)。
	>至於，未來遇到類似的事情，自己也算是有個解法了。
<br/>
3. 時間安排：事情的先後順序
	>只要一忙起來，就會忘記最初初衷目的是什麼，還有不斷加事情做的傾向(深怕錯過了什麼、什麼都想要的心態)，導致事情一堆，但都沒有明顯的加乘效果或後續效果。
	>如何把握原則? 時刻冷靜下來的有效方法、停止腦子不斷向未來想呢? 或許最有效的方法就是規劃安排時，保留一定的彈性時間。目前粗估，大概是可運用時間的1/3。

<br/>
上上學期：
<br/>
1. 我的思考與組織需要不斷地被逼到極限。  
    >不斷地問為什麼、構築完整具說服力的體系（一篇論文、一門技術）。我最快樂的時後是針對問題進行回答、根據他人的想法進行思考再組織思緒的過程，儘管並非所有人皆熱衷於討論，另外一個最快樂的時光就是思考論文相關的事情。不斷地問：這個問題如此回答是否有理？
    >這麼回答有沒有漏洞？這麼回答是否脈絡過於跳躍難以銜接？如果是給回饋的老師的話，會針對什麼提出質疑？  
2. 我需要回饋的理由是滿足感，一方面是來自針對疑惑在腦中得以持續整合思考的過程，另一方面是在得知能夠獲得回饋時、具有鼓舞力量的預期心理（我知道我可以做到，為此我必需盡我所能的逼自己到極限）。
3. 我需要深度思考感興趣主題的的個人空間與時間自由。
<br/>

## **課堂目標**
這學期開始，必修都修完了，所以選課很自由，但也因此拖到了開學加退選也還沒定下來。<br/>
主要是因為想要上很多課，像是日文寫作、口說或者歷史文學之類的通識課，後來看著血淋淋的行事曆，本來就很多預定的事情，所以硬是只修畢業必要的課程。這個過程反覆不定，現在開學第二週，還是在再多修一門語言課掙扎。<br/>
至於剩下修畢業必要之三門課定下來了，大概知道每門課的內容、授課風格，但一門關於中共的課，還抓不太到要怎麼樣好好運用聽課時間。我發現自己沒辦法專注在老師的講課上QQ 主要是因為不知道老師到底要根據哪個文本或主題來說。當然，這門課的主題涉及層面很廣(社會科學那裡不是這樣)，但這對本來對中共沒有太多理解的我來說，不好吸收學習。看起來要靠課外時間自己排進度系統性學習，至於聽課，可能當演講聽，複習讀到的東西。<br/>
其他一門是全英文的美國外交政策專題，一門是學術英文寫作。選擇專題課是想習慣英文聽力和口說，然後希望和中共的課相輔相成。專題課大致可以抓到接下來自己每週的學習方式、內容和方向，加上第二週就解決了個人口頭報告，也就輕鬆多了(大概?)。<br/>
至於學術英文寫作，就照表操課，跟著進度、作業練習和建構英文寫作的運作架構。<br/>
上學期希望在全英文課堂中口說發言，這學期就不特別要求這模糊、而且讓自己壓力很大的目標。目標就是好好上課、作筆記、思考記錄個人想法，課後課前複習預習，建構在特定主題的知識網。<br/>
如何增進英文能力，打算透過課外的背單字來解決根本問題。至於在這過程或者任何生活中遭遇的事情，先是懷抱著健康的認知與心態，再鼓勵自己一點一點勇敢踏入更深入的人際關係或者任何挑戰。<br/>
然後，最重要的目標是，每天睡足夠、睡好覺、吃好、心理安定，其他的才是其次。
