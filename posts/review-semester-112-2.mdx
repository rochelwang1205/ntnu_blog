## **學期回顧**

碩一結束了，這學期過得無比煎熬，除去選了四門課的因素，真正的原因到底是什麼，外部因素，我歸結於上學期為暖身期。課程的要求沒有如此高，還有，歷經一學期的洗禮，同學當初對研究所的新鮮感也減少了許多。內部因素從操作面來看，我變得過度貪心、取捨沒做好、沒有每週盤點和規劃，一堆事情要做，越到期末、越有一堆死線，層面包含了工作、課業報告、課業進度、程式課程的作業和專題。明明很清楚手頭處理的事情不超過三大項，但在缺乏定期盤點和規劃下，爆到至少6項以上的事務。在不同事務下，精神和專注力也被切割得很零碎，時間永遠不夠用，而死線一個接著一個來。
<br/>
對這學期想要達成的如下：

#1. 在全英文課程中的口說發言<br/>
#2. 藉由個人報告機會，建立和實踐研究流程及方法<br/>
#3. 額外的自主練習，推動個人成長(背單字、英文跟讀、參加學術活動…)<br/>
#4. 在課堂團體報告中多做點什麼
<br/>
關於這幾項目標，是非常模糊的，尤其是頭尾兩項。一方面撰寫的時間點剛好是在去完北京、內心還在震撼和滿頭思緒的時刻，一方面沒有明確的評估回饋機制，導致焦慮遠勝於踏實有效的執行。
<br/>
但在期末回顧時，也確實完成了很多事情，包含了自己一直想做的、對自己承諾過的事情，還有階段性須達成的目標。像是申請赴外交換，雖然因為日本沒有想交換的名額所以換成了中國；還有認識不同的人，因緣際會下去了趟山西，認識各種人，以及因為修課認識新同學；持續供稿給部落格XD；參加ccClub再來碰一遍Python，做個專題等等。
<br/>
可是，分配和放鬆紓壓的能力真的慘不忍睹，另外，各個項目的連結性不強導致學習成效破碎不連貫，這些後段再來說說。
<br/>
### 碩士期間幾大目標之實踐
在這學期實際上做了哪些事情來朝這些目標前進呢?<br/>
#1. 🔅學會生活、照顧自己：覺察自己的感受、拆解帶來感受的問題並客觀地思考、一步一步持續地練習、改變
	> 發生些不是那麼愉快的事情，概括來說包含感受到不被尊重、侮辱沒禮貌的行為等等人際關係的事情，有些事情當下沒辦法想開想通，但也因為自己堅持面對不舒服的情緒，然後採取行動，像是和信任的他人討論諮詢或者寫下日記，最後都有好好地以比較健康和放下的心態面對這些發生的事情。也因此，在後來也更快能解開理解自己情緒的癥結點，然後在下次能運用在過去的經驗修正。很開心自己確實做到了這點，有所成長。
<br/>
#2. 📌 建立自學模式 for 長期學習：學習策略、時間管理、閱讀理解與整合能力
	> 大致釐清了整個研究所、或者說目前生活狀態的遊戲規則了，而學習策略，也算是大抵摸清各種學習方法的重點，對於精進能力來說，也更明確能點出精進的方法和項目。因為半年內去中國三趟的密集經驗，特別在人際關係方面有諸多心得和想法。但這也導致如何選擇、選擇哪些才是核心的問題：第二季有很多事情，包含學校各課程的大報告、小報告、課外的前端課程、python專題，在這種情況下，而斷然的決定開始了新的工讀。用事後的角度來看，能去工讀真是太好了，但不是很想再次體驗這忙碌過頭的日常。可以做任何想做的事，但不是所有事情。取捨還是接下來要好好面對的課題。
<br/>
#3. 🧑‍🤝‍🧑open mind：be kind and friendly
	>心態相對於行動好像比較開方和友善。這一項目，是我最不清楚該怎麼衡量是否有開放心態、友善。只能說問心無愧嗎? 基本上都是問心無愧，但該說甚麼或不該說甚麼，看TPO這件事，蠻讓人糾結的，有好多次事後都覺得不該怎麼做......和人際關係相關。接下來要好好研究一番，既然個人已經有所基礎，那麼接下來就側重向外的部分了。
<br/>
#4. 💻硬實力：前端工程師
	>如同承諾，這學期報了六角學院的體驗營，達成最低目標，拿到證書和保持手感。也另外報了為期九個月的前端工程養成班。至於情況如何就待暑期回顧時細說。
<br/>

## **如果以這學期為單位，我的高光時刻(highlight)會是什麼呢？**

## **如果以這學期為單位，我的困境(difficulties)是什麼呢？**
#1. 臨場或有提前準備的舉手發言/發問
	>緊張、擔心問題不好、被嘲笑或對主題沒興趣
	>後來就用私下一對一的方式問
<br/>
#2. 被針對的感覺
	>總之，後來也用諸多的方式來化解這個不舒服的感覺，像是念課外書、思考(內耗??)。
	>至於，未來遇到類似的事情，自己也算是有個解法了。
<br/>
#3. 時間安排：事情的先後順序
	>只要一忙起來，就會忘記最初初衷目的是什麼，還有不斷加事情做的傾向(深怕錯過了什麼、什麼都想要的心態)，導致事情一堆，但都沒有明顯的加乘效果或後續效果。
	>如何把握原則? 時刻冷靜下來的有效方法、停止腦子不斷向未來想呢? 或許最有效的方法就是規劃安排時，保留一定的彈性時間。目前粗估，大概是可運用時間的1/3。
<br/>
## **總結**
完成這篇回顧時已是暑假尾巴、準備開學前的時候，所以在學期間所思所想或所經歷，其實因為暑假的經驗有了巨大的衝擊和體悟。如今看來，以上這些不管是好是壞、是正面的還是負面的、還是已知的或未知的，事實上依然要回到誠實面對自己。誠實地面對自己的情緒、價值觀、理念，然後回答那些看似尖銳卻直指核心的問題：關於人生的、工作職涯的、人際的、家庭的，至於他人的善意或惡意，就感謝那些善意、惡意就務實地、客觀地去處理吧。
